HTML이란?

Hypertext markup language (HTML) is the standard markup language for creating web pages and web applications. WEb browsers receive HTML documnets from a web server or from local storage and render them into multimedia web pages. HTML describes the structure of a web page semantically and originally included cues for the appearance of the document.
